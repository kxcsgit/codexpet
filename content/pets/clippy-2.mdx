# Clippy 2.0

"It looks like you're writing a resignation letter. Would you like help with that?"

The beloved (and hated) Microsoft Office assistant is back, living rent-free on your desktop once again.

## Features

- **Modern redesign** — vector graphics, smooth animations
- **Contextual tips** — actually useful (sometimes) suggestions
- **Nostalgia mode** — switch back to the original pixel art style
- **Easter eggs** — discover hidden reactions and dialogues
