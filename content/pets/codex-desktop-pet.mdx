# Codex Desktop Pet

The official desktop pet from OpenAI's Codex CLI — the AI coding assistant that took the developer world by storm.

## Features

- **Reactive animations** — dances when your code compiles, naps when idle
- **Status indicators** — shows build status, git branch, and more
- **Customizable skins** — community-created themes and appearances
- **Lightweight** — minimal CPU/memory footprint

## Platform Support

| Platform | Status |
|----------|--------|
| Windows  | ✅ Full support |
| macOS    | ✅ Full support |
| Linux    | ✅ Full support |

## How to Install

1. Install Codex CLI: `npm i -g @openai/codex`
2. Run `codex pet` to enable the desktop companion
3. The pet appears in your system tray and on your desktop
