# Shimeji

The original desktop pet experience. Shimeji is a Java-based desktop mascot system that has been charming users since 2009.

## Features

- **Window climbing** — characters crawl up and down your windows
- **Multiple characters** — run several mascots simultaneously
- **Community characters** — thousands of anime/game character skins
- **Physics-based** — realistic falling, sitting, and walking animations
- **Customizable** — create your own characters with sprite sheets

## Getting Started

1. Download the latest release from the GitHub page
2. Extract the archive
3. Run the Shimeji application
4. Choose your character and watch them explore your desktop

## Popular Characters

- Anime protagonists (Naruto, Goku, Luffy)
- Game characters (Mario, Kirby, Pikachu)
- OS-tans (Windows-chan, Mac-chan)
- Original community creations

## Legacy

Shimeji inspired an entire generation of desktop pets. Many modern desktop pet apps trace their lineage back to this classic Java application.
