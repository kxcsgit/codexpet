# Cursor Pet

The desktop companion for Cursor, the AI-first code editor. A tiny creature that sits beside your cursor and reacts to your coding sessions.

## Features

- **IDE integration** — understands your code context
- **Celebration dance** — when tests pass or PRs merge
- **Sleepy mode** — takes a nap when you're AFK
- **Pair programming vibe** — makes solo coding feel less lonely
